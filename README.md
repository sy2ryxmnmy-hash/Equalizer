# Equalizer APO Plugin R.A.T
this is an self coded base for an Equlizer Apo Plugin.
its based on [DearImGui](https://github.com/ocornut/imgui)Framework.

## Important Informations ❗️

This is for educational purposes only !
this repo don't exists to harm people just to show how it works.
The Project is coded in german so u have to rewrite the ui.

## Installation

1. make sure you have [Git](https://git-scm.com)installed at your Pc.
2. use following command in your terminal:
```bash
git clone https://github.com/sy2ryxmnmy-hash/Equalizer.git
```
3. now navigate to the folder and now you have succsesfully installed the project

## Documentation

It's pretty simple to code.
But you need to have knowledge about C++ and the ImGui Framework.
Just ignore the ext folder. There are the dependencies so ImGui works.
The workspace is the src folder. There is the coded stuff like base and ui/ux
to make the Rat just code in the [script.cpp](https://github.com/sy2ryxmnmy-hash/Equalizer/blob/main/src/script.cpp) file.
